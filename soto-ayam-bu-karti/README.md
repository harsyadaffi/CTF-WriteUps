# Soto Ayam Bu Karti (Binary Exploitation)

## Deskripsi
Challenge binary exploitation dengan sistem pembelian makanan.

## Analisis
Program memiliki fitur pembelian dengan input jumlah porsi.

## Vulnerability
Input negatif tidak divalidasi → menyebabkan saldo bertambah.

## Exploit
Masukkan nilai negatif untuk mendapatkan saldo besar, lalu beli flag.

## Flag
Hackfest0x09{skrrt_skrrt_skraa_skrraaa_skrrt_skrrrrttt}

## Tools
- Kali Linux
- netcat
